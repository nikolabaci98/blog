# blog
 
Simple blog website with no DB intergration.

Using NodeJS, ExpressJS, EJS in the server side and BootStrap, CSS and HTML in the client side.

Next steps:
1. Add DB
2. Add user authentication to allow:
     - login
     - edit/delete posts
     - follow other users
     - comment on posts
